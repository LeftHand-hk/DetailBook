# DetailBook - Setup Instructions

## Prerequisites

You need Node.js installed. Download it from: https://nodejs.org (LTS version)

## Quick Start

After installing Node.js, open a terminal in this folder and run:

```bash
npm install
npm run dev
```

Then open: http://localhost:3000

## Pages

| URL | Description |
|-----|-------------|
